How It Works

This demo uses Common Gateway Interface (CGI) to generate dynamic content. When you click the link above:

  1. The webserver receives your request
  2. It recognizes the request is for a CGI script
  3. It executes the script and captures its output
  4. The script generates a random inspirational quote
  5. The webserver sends the generated content back to your browser

Each time you refresh the page or click the button, a new quote is generated server-side.
