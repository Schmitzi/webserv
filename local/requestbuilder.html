How It Works

  • Method Blocks (blue): Define the HTTP method (GET, POST, DELETE)
  • Version Block (purple): Specify the HTTP version
  • Custom Blocks (orange): Add custom path, parameters, or headers
  • Once you've arranged your blocks, click Send Request to submit to the Webserv

Typical HTTP request format: METHOD /path HTTP/VERSION
